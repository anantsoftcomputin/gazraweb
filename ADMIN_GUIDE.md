## Managing Events

### Add New Event
1. Click "Events" in sidebar
2. Click "Add Event" button
3. Fill in event details:
   - **Title**: Event name
   - **Category**: theater, workshop, therapy, community, cultural
   - **Date**: Display date (e.g., "April 10, 2025")
   - **Time**: Event time (e.g., "8:00 PM")
   - **Location**: Venue details
   - **Capacity**: Number of attendees (e.g., "50 people")
   - **Description**: Full event description
   - **Price**: Ticket cost or "Free"
   - **External Link**: BookMyShow or registration link (optional)
   - **Featured**: Check to highlight on homepage
4. Upload event image (max 5MB)
5. Click "Create Event"

### Edit Event
1. Find event in list
2. Click "Edit" button
3. Modify details
4. Click "Update Event"

### Delete Event
1. Find event in list
2. Click "Delete" button
3. Confirm deletion

## Best Practices

### Images
- **Format**: JPG, PNG, or WebP
- **Size**: Under 5MB (compress before upload)
- **Dimensions**: Minimum 800x600px
- **Quality**: High resolution for events

### Event Descriptions
- Clear and concise
- Include what attendees will gain
- Mention any requirements
- Add contact for questions

### Regular Updates
- Update events weekly
- Review form submissions daily
- Archive past events monthly
- Keep menu current

## Troubleshooting

### Can't Log In
- Verify email and password
- Check internet connection
- Clear browser cache
- Contact technical admin

### Image Won't Upload
- Check file size (under 5MB)
- Verify file format (JPG, PNG, WebP)
- Try compressing image
- Check internet connection

### Changes Not Appearing
- Refresh browser (Ctrl/Cmd + R)
- Clear cache
- Wait 1-2 minutes for propagation
- Check if save was successful

### Error Messages
- Take screenshot of error
- Note what you were doing
- Try again after refreshing
- Contact technical support if persists

## Security Tips

### Password
- Use strong, unique password
- Don't share admin credentials
- Change password regularly
- Use password manager

### Logout
- Always logout when done
- Don't leave portal open on shared computers
- Close browser after logout

### Permissions
- Only authorized users should have access
- Don't create unnecessary admin accounts
- Report suspicious activity
